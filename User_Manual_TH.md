# 📖 คู่มือการใช้งานระบบแขนกลอัจฉริยะ (WebRobot) ฉบับเริ่มต้น (Beginner Guide)

ยินดีต้อนรับสู่โปรเจกต์ **WebRobot** (ระบบควบคุมแขนกล 4 แกน + ปากคีบ ผ่านหน้าเว็บ) คู่มือฉบับนี้ถูกเขียนขึ้นมาเพื่อให้ **คนที่ไม่มีพื้นฐานทางด้านโปรแกรมมิ่งหรือหุ่นยนต์เลย** สามารถอ่าน ทำความเข้าใจ และเปิดใช้งานระบบได้อย่างปลอดภัยทีละขั้นตอน!

---

## 1. 🐣 บทนำ (Introduction)
ระบบ WebRobot คืออะไร?
สั้นๆ คือเรามี **"แขนกลจำลอง"** ที่สามารถขยับซ้ายขวา ขึ้นลง หรือยืดหดได้คล้ายแขนคน และมี **"กล้อง"** เป็นดวงตาคอยมองหาวัตถุ (ไขควง, น็อต, ฯลฯ) 
ความเจ๋งคือเราไม่ได้บังคับมันด้วยรีโมท แต่ระบบจะใช้ **"สมองกล AI (YOLO)"** ช่วยมองวัตถุผ่านกล้อง และให้เรากดแค่คลิกเดียวบนหน้าเว็บ แขนกลก็จะวิ่งไปหยิบวัตถุนั้นๆ ได้เอง!

---

## 2. 📚 คำศัพท์เบื้องต้นที่ควรรู้ (Terminology)
ไม่ต้องกลัวศัพท์แสง มาทำความรู้จักคำเหล่านี้กันสั้นๆ:
- **Hardware (ฮาร์ดแวร์):** ชิ้นส่วนที่จับต้องได้ เช่น แผงวงจร มอเตอร์ สายไฟ แขนกล กล้อง
- **Software (ซอฟต์แวร์):** โปรแกรมที่มองไม่เห็น เช่น โค้ดที่สั่งงานให้หุ่นขยับ แผงควบคุมบนหน้าเว็บ
- **ESP32:** คือ "สมองส่วนกล้ามเนื้อ" เป็นแผงวงจรขนาดเล็กที่คอยรับคำสั่งไปขยับมอเตอร์
- **Servo Motor (เซอร์โวมอเตอร์):** กล้ามเนื้อของแขนกล เป็นมอเตอร์ที่ขยับหมุนไปตามองศาที่เราสั่ง
- **PCA9685:** โมดูลพี่เลี้ยง คอยช่วย ESP32 คุมมอเตอร์หลายๆ ตัวพร้อมกัน เพื่อไม่ให้ไฟตก
- **YOLO (YOLOv11 / YOLO26):** คือ "สมองส่วนตา" เป็น AI ที่ถูกสอนมาให้จำแนกหน้าตาของวัตถุ ว่าอันนี้คือน็อต อันนี้คือไขควง
- **COM Port:** ช่องทางเสียบสาย USB บนคอมพิวเตอร์ของคุณ (เช่น เมื่อเสียบสายบอร์ด ESP32 คอมพิวเตอร์จะจำว่าเป็นช่อง COM3 หรือ COM4)

---

## 3. 🔌 การเตรียมและระวังเรื่องสายไฟ (Hardware & Wiring)
**⚠️ เรื่องนี้สำคัญมากที่สุด: ป้องกันบอร์ดไหม้**
มอเตอร์ของแขนกล (Servo) ใช้พลังงานเยอะมาก! หากเราเอาแผงวงจรจิ๋ว (ESP32) ไปจ่ายไฟให้มันตรงๆ แผงวงจรจะไฟตกและรีสตาร์ทตัวเอง หรือร้ายแรงคือ "ไหม้" ทันที
1. **แยกสายไฟเลี้ยงให้ชัดเจน:** 
   - ให้เสียบสาย USB ของแผง ESP32 เข้ากับคอมพิวเตอร์ (เพื่อส่งคำสั่ง) 
   - และ **ต้องมี Adaptor แหล่งจ่ายไฟ 5V แยกต่างหาก** เสียบเข้ากับ ขั้ว V+ ของแผงวงจร PCA9685 เท่านั้น ห้ามนำไฟจากขั้ว 5V ของบอร์ด ESP32 มาใช้เลี้ยงมอเตอร์เด็ดขาด!
2. **การติดตั้งกล้อง:** เสียบกล้อง USB Webcam เข้ากับคอมพิวเตอร์ จัดตำแหน่งให้กล้องส่องลงมาจากมุมสูง (Top-Down) มองเห็นพื้นที่ที่จะให้แขนกลหยิบของ

## 4. 💻 การเตรียมโปรแกรมพื้นฐาน (Software Setup)
ในคอมพิวเตอร์ที่จะรันระบบ ต้องติดตั้ง 3 โปรแกรมหลักๆ ดังนี้ก่อน (ทำแค่ครั้งแรกครั้งเดียว):
1. **Arduino IDE:** โปรแกรมสำหรับฝังโค้ด (Upload) ภาษา C++ ลงไปในสมอง ESP32
2. **Python:** ภาษาโปรแกรมสำหรับเปิดระบบ Server (ระบบหลังบ้าน) และรัน AI กล้อง
3. **Node.js:** โปรแกรมตัวช่วยสำหรับเปิดหน้าเว็บแผงควบคุม (Dashboard)

*(วิธีติดตั้งสามารถค้นหาใน Google หรือ YouTube ได้เลย ใช้เวลาติดตั้งเพียง 5 นาที)*

## 5. 🚀 วิธีเปิดใช้งานระบบ (Step-by-step)
เราจะเริ่มเปิดระบบจาก "กล้ามเนื้อ" > "สมอง" > "หน้าจอควบคุม"

### ขั้นตอนที่ 5.1: สั่งงานสมองกล (Flash ESP32)
1. เปิดโปรแกรม Arduino IDE ขึ้นมา
2. เปิดไฟล์ `robot_arm_4dof_final (1).ino` ที่อยู่ในโฟลเดอร์ `esp32`
3. ไปที่เมนู Tools > Port เลือก COM Port ที่ตรงกับสาย USB ของ ESP32
4. กดปุ่มลูกศร 👉 **"Upload"** เพื่อฝังโค้ด รอจนเสร็จแถบเขียว 

### ขั้นตอนที่ 5.2: เปิดระบบหลังบ้านและระบบ AI กล้อง (Server)
ระบบนี้ทำงานเป็นตัวกลางในการดูกล้อง และสั่งการ ESP32
1. ให้เปิดโปรแกรม `Command Prompt` (Terminal/CMD) ในคอมพิวเตอร์
2. พิมพ์คำสั่งเพื่อเข้าไปในโฟลเดอร์ `server` (เช่น `cd ลงไปที่อยู่โฟลเดอร์ WebRobot/server`)
3. พิมพ์เพื่อติดตั้งเครื่องมือเสริม (ทำแค่ครั้งแรกครั้งเดียว): `pip install -r requirements.txt`
4. **เปิดการทำงาน:** พิมพ์ `python app.py` 
5. ถ้าระบบขึ้นว่า "Application run on http://localhost:5000" แปลว่าพร้อมแล้ว! (ห้ามปิดหน้าต่างสีดำหน้านี้ ให้พับลงไป)

### ขั้นตอนที่ 5.3: เปิดหน้าเว็บควบคุม (Web Client)
หน้าจอนี้คือส่วนที่คุณจะใช้งานจริง!
1. เปิด `Command Prompt` ขึ้นมาอีก 1 หน้าต่าง
2. เข้าไปในโฟลเดอร์ `client`
3. พิมพ์: `npm install` (ทำครั้งแรกครั้งเดียว)
4. **เปิดหน้าเว็บ:** พิมพ์ `npm run dev`
5. นำลิงก์ `http://localhost:5173` ไปแปะเปิดใน Google Chrome หรือเบราว์เซอร์อื่นๆ

## 6. 🎮 วิธีใช้งานหน้าเว็บควบคุม (Web Dashboard Guide)

เมื่อเข้ามาในหน้าเว็บแล้ว จะเจอปุ่มและหน้าจอภาพ ให้ทำตามลำดับนี้:

### 1) การเชื่อมต่อกับแขนกล
มองหาเมนู "Port / เครื่องมือการเชื่อมต่อ" จะมีรายชื่อ COM Port โผล่ขึ้นมา (อันเดียวกับใน Arduino IDE)
ให้กดเลือกและกดปุ่ม **[Connect]** เมื่อเชื่อมต่อสำเร็จจะขึ้นสถานะ Ready หรือ สีเขียว

### 2) การเปิดกล้อง
มองหาปุ่ม "Camera / กล้อง" ให้เลือกหมายเลขกล้อง (เช่น Camera 0 หรือ 1) แล้วกด **[Start]**
รอสักพักภาพจากบนโต๊ะของคุณจะมาโผล่บนหน้าเว็บ

### 3) การปรับพิกัดให้รู้จักโลกความจริง (ArUco Calibration) 📍
*ขั้นตอนนี้สำคัญมาก เพื่อให้ AI รู้ว่าแขนกลอยู่ที่ไหน*
1. นำกระดาษที่มีแผ่นขาวดำทรงสี่เหลี่ยม (เรียกว่า ArUco Marker) 4 มุม มาวางบนโต๊ะในรัศมีที่แขนกลก้มถึง
2. กดปุ่ม **[Calibrate]** เบาๆ 1 ครั้ง
3. หากกล้องมองเห็นทั้ง 4 มุมชัดเจน จะมีกรอบพื้นที่ล้อมรอบขึ้นมา แปลว่าระบบรู้พิกัดระยะทางบนโต๊ะแล้ว!

### 4) การควบคุมมือ (Manual Mode)
- **แถบเลื่อน X, Y, Z:** คุณสามารถลากขวาซ้ายเพื่อบังคับแขนยืดหด (X), ขยับซ้ายขวา (Y), ก้มเงย (Z) แบบจอยสติ๊กได้เลย
- **ปุ่ม HOME 🏠:** เมื่อแขนกลทำท่าแปลกๆ หรือพันกัน ให้กดปุ่มนี้ แขนและข้อศอกจะยกกลับมาตั้งฉาก 90 องศาอย่างปลอดภัย
- **ปุ่ม GRAB (คีบ) / RELEASE (ปล่อย):** ใช้กดคีบปากคีบให้หนีบหรืออ้าออก

### 5) 🤖 โหมด AI สั่งหยิบอัตโนมัติ (Pick & Place)
- เมื่อนำวัตถุ (ไขควง/กล่อง) ไปวางใต้กล้อง หน้าจอจะมี **กรอบสี่เหลี่ยมคลุมวัตถุ (Bounding Box)** พร้อมชื่อขึ้นมาโดยอัตโนมัติ!
- หากต้องการให้แขนกลหยิบ ให้คุณ "คลิกที่รายชื่อวัตถุนั้น" บนแถบหน้าต่างของเว็บ
- ปล่อยมือได้เลย! แขนกลจะค่อยๆ หมุนไปยังเลนกึ่งกลางวัตถุ ลดระดับลง คีบ ดึงขึ้นมา และนำไปวางที่เป้าหมายถัดไปให้เอง

## 7. 🛠️ โปรแกรมของแถม (สำหรับตรวจสอบก่อนลงสนามจริง)

บางครั้งเราอยากลองขยับก่อน หรือทดสอบกล้องว่ามองเห็นของบนโต๊ะชัดไหม เราแถมไฟลพิเศษ 2 ตัวมาให้ในโฟลเดอร์หลักครับ!

1. **โปรแกรมจำลองแขน 3D (`robot_arm_sim.py`)** 
   - พิมพ์ใน Terminal ว่า `python robot_arm_sim.py`
   - มันจะเด้งโปรแกรมจำลองโมเดลแขนกลเป็นเส้นกราฟิก 3 มิติ ขึ้นมาบนหน้าจอคอมพิวเตอร์ ให้คุณกะระยะ X, Y, Z โดยที่แขนจริงยังไม่ขยับ ปลอดภัยมาก!
2. **โปรแกรมทดสอบตาของ AI (`test_yolo26.ipynb`)**
   - พิมพ์ใน Terminal ว่า `jupyter notebook`
   - ลองนำรูปภาพมาทดสอบดูว่า AI โมเดลเวอร์ชั่นใหม่ล่าสุด (YOLO26) มองเห็นไขควงเราชัดรึเปล่า โดยไม่ต้องเปิดกล้องจริงๆ
