# 📊 WebRobot: Slide Presentation Guide
*(เนื้อหาสำหรับก๊อปปี้ลงสไลด์นำเสนอ แบ่งเป็นหน้าต่างๆ)*

---

### **Slide 1: Title**
**หัวข้อ:** WebRobot: ระบบควบคุมแขนกลอัจฉริยะ (AI Pick & Place)
**หัวข้อย่อย:** การสั่งงานด้วย AI (YOLO) ข้าม Web Dashboard และ ESP32
**เนื้อหา/ภาพแนะนำ:** ภาพรวมของระบบ (แขนกล + หน้าเว็บ + กล้อง Webcam)

---

### **Slide 2: System Overview (ภาพรวมระบบ)**
**หัวข้อ:** สถาปัตยกรรมระบบ 3 ส่วนหลักทำงานร่วมกัน
- **1. Web Dashboard (React):** หน้าจอสั่งงานง่ายๆ ผ่านเบราว์เซอร์
- **2. AI Server (FastAPI):** เซิร์ฟเวอร์หลังบ้านประมวลผลตา AI (YOLO26)
- **3. Microcontroller (ESP32):** สมองกลควบคุมกล้ามเนื้อ (Servo) แกนต่างๆ

---

### **Slide 3: Hardware Preparation (การเตรียมฮาร์ดแวร์)**
**หัวข้อ:** ข้อควรระวังและการต่อวงจร
- **แยกไฟเลี้ยงชัดเจน (สำคัญมาก!):** 
  - 🔌 เสียบ USB ที่บอร์ด ESP32 เพื่อรับ/ส่งข้อมูล
  - ⚡ จ่ายแหล่งไฟนอก (5V) เข้าที่โมดูล PCA9685 เท่านั้น
- **ข้อห้าม:** ห้ามดึงไฟ 5V จาก ESP32 ไปเลี้ยงมอเตอร์เด็ดขาด (บอร์ดอาจไหม้)
- **การตั้งกล้อง:** มุมสูง (Top-Down) มองลงมาที่หน้าโต๊ะทำงาน

---

### **Slide 4: System Startup (วิธีเปิดใช้งาน)**
**หัวข้อ:** 3 ขั้นตอนการเปิดระบบแบบง่าย
- **Step 1:** จ่ายไฟ และเสียบสาย USB แขนกลเข้าคอมพิวเตอร์
- **Step 2:** เปิดเซิร์ฟเวอร์หลังบ้าน (AI)
  - `cd server` ➡️ พิมพ์รัน `python app.py` 
- **Step 3:** เปิดหน้าจอสั่งการ (Web Dashboard)
  - `cd client` ➡️ พิมพ์รัน `npm run dev` ➡️ เปิดลิงก์ `localhost:5173`

---

### **Slide 5: Dashboard - Getting Started**
**หัวข้อ:** เริ่มต้นเชื่อมต่อจากหน้าเว็บ
- **1. เชื่อมต่อแขนกล (Connect):** เลือกช่อง COM Port แล้วกดปุ่ม *Connect* ให้ขึ้นไฟเขียว Ready
- **2. เปิดตาวิเศษ (Camera):** เลือกหมายเลขกล้อง Webcam แล้วกด *Start* เพื่อรับภาพสด
- **3. อัปเดตพิกัด (Calibration):** 
  - วางกระดาษ ArUco Marker 4 มุม
  - กด *Calibrate* เพื่อให้ระบบสร้างแผนที่ 3D บนโต๊ะจริง

---

### **Slide 6: Manual Control (โหมดบังคับมือ)**
**หัวข้อ:** ควบคุมอิสระด้วย XYZ Sliders
- **Joystick แกน X, Y, Z:** ใช้นิ้วลาก/เลื่อนแถบเพื่อยืดแขน, ขยับซ้ายขวา, หรือยกขึ้นลง
- **🏠 ปุ่ม HOME:** รีเซ็ตแขนกลับท่าตั้งฉาก 90 องศาอย่างปลอดภัยเมื่อฉุกเฉิน
- **✊ ปุ่ม GRIPPER:** สั่งหนีบ (Grab) หรือปล่อย (Release) ปากคีบได้ทันที

---

### **Slide 7: AI Pick & Place (โหมดอัตโนมัติ)**
**หัวข้อ:** หยิบจับอัจฉริยะด้วย AI (YOLO26)
- **การตรวจจับ (Detection):** นำวัตถุไปวางใต้กล้อง AI จะตีกรอบสี่เหลี่ยม (Bounding Box) ให้ทันที
- **สั่งการง่ายๆ (1-Click):**
  - คลิกที่ "ชื่อวัตถุ" ที่โชว์ขึ้นบนหน้าจอ
  - แขนกลจะแปลงพิกัด (IK) เลื่อนไปคร่อมวัตถุ คีบ และนำไปวางที่เป้าหมายให้เอง 🤖✨

---

### **Slide 8: Simulation Tool (ของแถมพิเศษ)**
**หัวข้อ:** ทดสอบจำลองก่อนลงสนามจริง (Safe Mode)
- **3D Simulator (`robot_arm_sim.py`):**
  - โปรแกรมจำลอง 3 มิติ เพื่อเช็คพิกัดแขนกลก่อนของจริงขยับ
- **Jupyter Test (`test_yolo26.ipynb`):**
  - ทดสอบระบบวิเคราะห์ภาพของ YOLO ด้วยรูปล่วงหน้า โดยไม่ต้องรันเซิร์ฟเวอร์เต็มรูปแบบ
